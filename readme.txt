"GAL-V1-R1"

This is an MX keyboard for the Galaksija Plus computer.
It uses a 6.25u spacebar and is close to, but modified
compared to the standard layout in order to use modern
PC-type MX keycaps.

Included are the Kicad 7 design files.
Gerbers are included that can be sent to the board
house.
